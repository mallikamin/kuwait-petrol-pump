## Issues Reported by User

### 1. ❌ Meter Readings: Shift Selection Error
```
/api/shifts/active:1  Failed to load resource: 404
/api/shifts/active:1  Failed to load resource: 400
```

**Root Cause**: Frontend calling wrong endpoint
- Frontend was calling: `GET /api/shifts/active`
- Backend endpoint is: `GET /api/shifts/current?branchId=X`

### 2. ❌ POS: Select.Item Error
```
Error: A <Select.Item /> must have a value prop that is not an empty string.
```

**Root Cause**: Customer dropdown using empty string value
- Code had: `<SelectItem value="">Walk-in customer</SelectItem>`
- React Select doesn't allow `value=""`

## Fixes Applied

### Fix 1: Meter Readings Shift API ✅

**Changed in `apps/web/src/api/shifts.ts`:**
```typescript
// BEFORE
getActive: async (): Promise<Shift[]> => {
  const response = await apiClient.get<Shift[]>('/api/shifts/active');
  return response.data;
},

// AFTER
getCurrent: async (branchId: string): Promise<Shift | null> => {
  try {
    const response = await apiClient.get<{ currentShift: Shift | null }>('/api/shifts/current', {
      params: { branchId },
    });
    return response.data.currentShift;
  } catch (error) {
    return null;
  }
},
```

**Changed in `apps/web/src/pages/MeterReadings.tsx`:**
```typescript
// BEFORE
const { data: activeShifts } = useQuery({
  queryKey: ['shifts', 'active'],
  queryFn: () => shiftsApi.getActive(),
});

// AFTER
const { data: currentShift } = useQuery({
  queryKey: ['shifts', 'current', branchId],
  queryFn: () => branchId ? shiftsApi.getCurrent(branchId) : Promise.resolve(null),
  enabled: !!branchId,
});
```

**UI Change:**
- **Before**: Dropdown with list of shifts (array)
- **After**: Read-only display of current shift (single object)

### Fix 2: POS Customer Dropdown ✅

**Changed in `apps/web/src/pages/POS.tsx`:**
```typescript
// BEFORE
<SelectItem value="">Walk-in customer</SelectItem>

// AFTER
<SelectItem value="none">Walk-in customer</SelectItem>
```

**Also updated logic:**
```typescript
// selectedCustomer calculation
const selectedCustomer = selectedCustomerId && selectedCustomerId !== 'none'
  ? customers.find(c => c.id === selectedCustomerId)
  : undefined;

// Sale payload
customerId: selectedCustomerId && selectedCustomerId !== 'none' ? selectedCustomerId : undefined,
```
