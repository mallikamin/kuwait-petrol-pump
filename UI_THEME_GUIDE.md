## 🌍 Urdu Language UI/UX Considerations

### **Right-to-Left (RTL) Layout**
- Navigation items order: flip horizontally
- Table headers: right-aligned
- Text: Right-to-left reading order
- Borders: Change left borders to right borders
- Buttons: Mirror layout

### **Font Recommendations for Urdu:**
```css
font-family: 'Segoe UI', 'Urdu Typesetting', 'Arial Unicode MS', sans-serif;
```

### **Urdu Text Rendering:**
- Use minimum font size 14px (Urdu needs more vertical space)
- Increase line-height to 1.6-1.8
- Ensure adequate padding around text
- Use high contrast for readability

### **Keyboard & Input:**
- Support Urdu keyboard layouts
- Allow users to toggle language
- Display numerals in Urdu (۰-۹) or English based on preference
- Placeholder text in Urdu

### **Color Considerations for Urdu Users:**
- Green: Positive, trusted (Pakistan flag color)
- Gold: Premium, formal (Islamic heritage)
- Blue: Professional, reliable
- Red/Orange: Warnings, energy
- Avoid excessive animations (accessibility)

## 🛠️ Technical Implementation Notes

### **Tailwind CSS Configuration:**
```javascript
theme: {
  colors: {
    // Modern Blue (default)
    primary: '#2196F3',
    dark: '#1565c0',

    // Can extend with theme-specific colors
    gold: '#D4AF37',
    teal: '#1a5f5f',
    'pak-green': '#1B5E20',
  }
}
```

### **Theme Switching:**
```javascript
// Store user theme preference
localStorage.setItem('userTheme', 'modern-blue');

// Apply theme class
document.body.classList.add(`theme-${theme}`);
```

### **RTL Support:**
```css
[dir="rtl"] {
  direction: rtl;
  text-align: right;
}

[dir="rtl"] .sidebar {
  right: 0;
  border-right: none;
  border-left: 1px solid #ddd;
}
```
